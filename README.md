# Dolet Syntax

VS Code syntax highlighting for the **Dolet** language.

## Features
- Keywords, types, numbers
- Strings + interpolated strings (`@"..."` with `{expr}`)
- Comments: `#` and triple-quoted `""" ... """`
- Functions/classes/structs/enums
- Maps/Lists and operators

## Install (Dev)
- Open this folder in VS Code
- Press `F5` to launch Extension Development Host
- Open a `.dlt` file to see highlighting
